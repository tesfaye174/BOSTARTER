BOSTARTER Core Systems Documentation

This documentation describes the centralized core systems implemented to improve code organization, reduce duplication, and standardize functionality across the BOSTARTER platform.

Note: Legacy files (animations.js, notifications.js, etc.) are now deprecated. Please use these centralized systems instead.

Quick Start

To use the core systems in your HTML files:

<script src="js/core/loader.js" type="module"></script>

This single import provides access to all core systems through the global window.BOSTARTER namespace.

Systems Overview

NotificationSystem

Centralized notification management for consistent user alerts.

Usage Examples:

// Show success notification
NotificationSystem.success('Operation completed successfully');

// Show error notification
NotificationSystem.error('An error occurred');

// Show warning with custom duration
NotificationSystem.warning('Connection unstable', { duration: 10000 });

// Show info notification
NotificationSystem.info('New updates available');

AnimationSystem

Centralized animation management for consistent UI transitions.

Usage Examples:

// Animate a single element
AnimationSystem.animate(element, 'fadeIn');

// Animate multiple elements with stagger
AnimationSystem.stagger(elements, 'slideInRight', { staggerDelay: 100 });

// Setup scroll animations
AnimationSystem.setupScrollAnimations('.animate-on-scroll', 'fadeIn');

// Animate progress bars
AnimationSystem.animateProgressBars('.progress-bar');

Utils

Enhanced utility functions for common operations.

Usage Examples:

// Format date
Utils.formatDate(new Date(), { year: 'numeric', month: 'long', day: 'numeric' });

// Format relative time
Utils.formatRelativeTime(timestamp); // "2 days ago"

// Format currency
Utils.formatCurrency(1599.99); // "€1,599.99"

// Format number with compact notation
Utils.formatCompactNumber(1500); // "1.5K"

// Calculate progress percentage
Utils.calculateProgress(75, 100); // 75

// Escape HTML content
Utils.escapeHtml('<script>alert("XSS")</script>');

Migration Guide

From Standalone Notifications

// Old approach
showNotification('Message', 'success');

// New approach
NotificationSystem.success('Message');

From Custom Animations

// Old approach
element.style.opacity = '0';
element.style.transform = 'translateY(20px)';
setTimeout(() => {
    element.style.transition = 'all 0.6s ease';
    element.style.opacity = '1';
    element.style.transform = 'translateY(0)';
}, delay);

// New approach
AnimationSystem.animate(element, 'fadeIn', { 
    duration: 600,
    delay: delay
});

From Direct Utility Functions

// Old approach
function formatNumber(num) {
    return num.toLocaleString();
}

// New approach
Utils.formatNumber(num);
